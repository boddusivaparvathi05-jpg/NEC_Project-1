# Customer AI Project

This folder contains a starter structure for a customer analytics project with:
- synthetic dataset generation
- preprocessing utilities
- churn prediction and recommendation components
- a simple app entry point

## Suggested workflow
1. Install dependencies: pip install -r requirements.txt
2. Generate data: python dataset_generator.py
3. Train models: python model.py && python churn_model.py
4. Launch app: streamlit run app.py
